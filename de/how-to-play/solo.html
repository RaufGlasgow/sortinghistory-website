So spielen Sie Solo

Solo ist die Standard-Spielart von Sorting History. Waehlen Sie Ihre Kategorien, stellen Sie den Schwierigkeitsgrad ein und bauen Sie Ihre Zeitleiste in Ihrem eigenen Tempo auf. Speichern Sie mittendrin und kommen Sie spaeter zurueck, wenn das Leben dazwischenkommt.

Ein Solo-Spiel einrichten

  1. Tippen Sie auf Spielen im Hauptmenue.
  2. Waehlen Sie Ihren Spielmodus:
    • Sorting History — Ereignisse in chronologischer Reihenfolge auf einer Zeitleiste platzieren
    • History Pinpoint — das genaue Jahr eines Ereignisses erraten (Historian-Stufe)
  3. Waehlen Sie Solo als Spielart.
  4. Waehlen Sie Ihre Kategorien. Eine fuer ein fokussiertes Spiel oder mehrere fuer Abwechslung. Verwenden Sie Voreinstellungen wie "Kriege & Politik" oder "Popkultur" fuer eine schnelle Einrichtung, oder speichern Sie Ihre eigenen Lieblingskombinationen.
  5. Stellen Sie den Wertungsmodus ein:
    • Freundlich — +2 fuer richtig, keine Strafe fuer falsch. Ideal zum Lernen.
    • Wettbewerbsorientiert — +2 fuer richtig, -1 fuer falsch. Ausgewogene Herausforderung.
    • Experte — +3 fuer richtig, -2 fuer falsch. Jede Platzierung zaehlt.
  6. Jahresbasierte Wertung umschalten (optional). Standardmaessig AN — Ereignisse im selben Jahr sind austauschbar. Schalten Sie sie AUS fuer eine haertere Herausforderung, bei der Ereignisse im selben Jahr in exakter Monatsreihenfolge platziert werden muessen.
  7. Spiellaenge waehlen:
    Laenge Runden Dauer
    Sprint5~3 Min.
    Normal10~5 Min.
    Marathon15~8 Min.
    Ultra-Marathon20~12 Min.
    Episch30~18 Min.
  8. Tippen Sie auf Start.

Waehrend des Spiels

Jede Runde erscheint eine Ereigniskarte mit Titel und Beschreibung. Ihre Aufgabe haengt vom Modus ab:

  • Sorting History: Platzieren Sie das Ereignis an der richtigen Stelle auf der wachsenden Zeitleiste. Setzen Sie es vor, nach oder zwischen bestehende Ereignisse, je nachdem wann es passierte.
  • History Pinpoint: Erraten Sie das Jahr mit intuitiven Wischgesten, den Frueher- und Spaeter-Buttons, oder tippen Sie auf die Jahresanzeige, um direkt einzugeben. Epochen-Chips springen zu breiten Zeitraeumen.

Nach jeder Platzierung oder Schaetzung zeigt das Spiel die Antwort und ob Sie richtig lagen.

Speichern & Fortsetzen

Muessen Sie mittendrin aufhoeren? Solo-Spiele werden automatisch gespeichert. Eine Schaltflaeche Spiel fortsetzen erscheint im Hauptmenue, wenn Sie ein gespeichertes Spiel haben. Sie setzen genau an der Runde fort, in der Sie aufgehoert haben.

Nach dem Spiel

Wenn alle Runden abgeschlossen sind, zeigt der Ergebnisbildschirm:

  • Ihre Punktzahl — insgesamt verdiente Punkte
  • Genauigkeit — Prozentsatz korrekter Platzierungen
  • Buchstabennote — von S+ (beste) bis F
  • Kategorie und benoetigte Zeit

Details zur Funktionsweise von Noten und Punktevergabe finden Sie unter Tipps, Strategie und Punktevergabe.

Empfehlung fuer das erste Spiel

Wenn Sie noch nie gespielt haben:

  • Modus: Sorting History
  • Spielart: Solo
  • Kategorie: Eine, die Sie gut kennen (Musik & Unterhaltung ist ein Publikumsliebling)
  • Wertung: Freundlich (keine Strafen)
  • Laenge: Sprint (5 Runden, unter 3 Minuten)

So bekommen Sie ein schnelles Gefuehl fuer die Spielmechanik ohne Druck. Sobald Sie sich wohlfuehlen, probieren Sie den Wettbewerbsorientierten Modus, laengere Spiele oder mischen Sie mehrere Kategorien.
