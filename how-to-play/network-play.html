How to Play Network Play

Network Play lets each team play on their own device, connected over the same WiFi network. One person hosts, everyone else joins. No internet required.

What You Need

  • 2–8 iOS devices (iPhones, iPads, or a mix)
  • Sorting History installed on every device
  • Same WiFi network — all devices must be connected to the same WiFi. No internet is needed, just a shared local network.

For the Host

The host creates the game and controls the settings. Any device can be the host.

  1. Tap Play on the main menu.
  2. Select Sorting History as the game mode.
  3. Choose Network Play and tap Host Game.
  4. Set up teams. Add 2–8 teams with custom names.
  5. Pick categories, scoring mode, and game length — same options as Solo and Pass & Play.
  6. Wait for guests to connect. The host screen shows which devices have joined.
  7. Tap Start Game once everyone is connected.

For Guests

  1. Make sure you're on the same WiFi network as the host.
  2. Tap Play on the main menu.
  3. Select Sorting History as the game mode.
  4. Choose Network Play and tap Join Game.
  5. Wait for the host. You'll see "Waiting for host..." until the game begins.

During the Game

  • Teams take turns in a shared rotation. When it's your team's turn, your device becomes active.
  • Place your event on the timeline, just like Solo or Pass & Play.
  • Results are shared. Each placement is broadcast to all connected devices so everyone sees the outcome.
  • Disconnections are handled. If a device drops off the network, the game pauses until it reconnects.

Network Play vs. Pass & Play

Pass & Play Network Play
Devices 1 (shared) 1 per team
How teams play Take turns, pass the device Each team on their own device
Connection None (fully offline) Same WiFi (no internet needed)
Best for Families, travel, quick setup Parties, competitive groups, larger gatherings

Troubleshooting

  • "Can't find host" — Make sure both devices are on the exact same WiFi network (not one on WiFi and one on cellular). Try toggling WiFi off and on.
  • Slow connection — Network Play uses local WiFi, so distance from the router matters. Stay in the same room for best results.
  • Device disconnected mid-game — The game pauses. Reconnect to WiFi and the game should resume. If the device can't reconnect, the game continues without that team.
